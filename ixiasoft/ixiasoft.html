DITA Language

  • shortdesc
    • This element represents the purpose or theme of the topic, also intended to be used as a link preview and for search reference.
  • metadata
    • All information related to the topic being worked on
  • indexterm
    • This element is used to produce an index entry in a generated index. You can nest indexterm elements to create multi-level indexes. The content is not output as part of the topic content, only as part of an index.
