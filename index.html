Type of Manuals

  • Operator's Manuals: Manuals for the drivers
  • Service Manuals: Repair manuals for technicians
  • Programming Guides: Computer-parameter manuals for technicians

Type of Topics

  • Concept: Used for information (definition/description of components)
  • Task: Used for procedures (removals, disassemblies, and maintenance)
  • Troubleshooting: Used to trobleshoot problems of systems/components
  • Reference: Used for data and specifications of components (component location and specifications)
  • Glossentry: Used for glossary
  • Referable Content: Used for reusable content (only the IA can use it)
  • Topic: Not used

DITA Language

Short-Description: Brief description to describe the topic

Metadata: All information related to the topic being worked on

Body: It is the body of the whole topic

P: These are the tags where most of text should go
